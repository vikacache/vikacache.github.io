MatScale

Texture & material upscaling, entirely inside the Unity Editor.

MatScale is an editor-only extension. Select texture assets, materials, or renderer GameObjects, queue 2× or 3× upscales, and the processing runs entirely on your machine - no cloud, no external services, no package dependencies. Sources are never modified; results are written as new lossless files beside them.

Before a job runs, the window shows exactly what you'll get: the generated resolution, the predicted file size, the processing route, and the active runtime - so there are no surprises on large textures.

Editor-only
Opens from Tools → MatScale. Nothing is added to your builds or runtime.
100% local Bundles open-source components: an upscaling model (CC BY 4.0), ONNX Runtime (MIT) and DirectML. Full third-party notices ship inside the package.
Inference runs on your hardware - GPU-accelerated on Windows x64 and Apple Silicon, with CPU fallback everywhere else.
2× and 3× output
1024 → 2048 or 3072 from one queued job.
Tiled & queued
Large textures are processed in tiles through a sequential queue, so the editor stays responsive.
Non-destructive
Writes lossless .tga files to an Upscaled folder next to the source. Originals are untouched.
Textures, materials & renderers
Upscale a single map, a material's texture slots, or every material on a selected renderer.

About the publisher

Vikacache is an independent Unity tools publisher with five years of development experience across editor tooling and game projects. Every asset is built from both sides of the workflow: as the package developer who maintains it, and as the game developer who has to live with it inside a production project.

That perspective sets the bar for everything we ship - tools should remove a real pain point, work on first import without setup rituals, and hold up against production content, not demo scenes.
